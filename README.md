# QWK - landing page

few steps requires to run this application

# Steps

1.Clone the application

```sh
$ git clone https://github.com/praveenganesh/paypay.git
```

2.Install the node modules

```sh
$ npm install
```

3.Run the application

```sh
$ npm start
```

Application will run at

```sh
http://localhost:3000/
```
